### Installation

1. Clone the reposetory
   ```sh
   git clone https://github.com/PeterHinge/README-templates.git
   ```
2. Install Python
3. Install Django
   ```sh
   pip install django
   ```
4. Start the server 
   ```sh
   python manage.py runserver
   ```
